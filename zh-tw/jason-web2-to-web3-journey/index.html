技術後端向 Jason：資深 Web2 開發者的轉型之路

約 1295 字大约 4 分鐘

2025-06-15

在本章，我們追溯 Jason 從大廠 C/C++ 開發者轉向 Web3 的路徑：研二自建以太坊節點、內推與線下社群獲取崗位，靠多段實習與紮實底層基礎拿下頂級交易所面試；強調 CPU/IO 原理對鏈上系統最佳化的必要性，指出前端遷移最易、底層開發需補產品思維；建議小白按專業優勢切入，實踐—補基礎形成閉環，牢記理解業務才是 AI 難以替代的核心競爭力。

Jason，現任某網際網路大廠 C/C++ 開發，深大碩士畢業，曾斬獲阿里、華為全職 Offer，推特 @fxxkmystyle

1. 我們邀請到國內某頭部網際網路公司開發工程師 Jason 老師，就他從 Web2 到 Web3 轉型之路進行分享，首先請 Jason 老師做一個簡單的自我介紹。

大家好，我叫 Jason，2023 年研究生畢業，本科、碩士都學計算機，碩士階段主攻計算機交叉學科。研二時跑過自己的以太坊節點；再往前，2016 – 2017 年上大學時就對虛擬貨幣有過淺嘗。現在主業寫 C/C++，偶爾寫 Android、iOS 以及 HarmonyOS 相關程式碼，正在把重心逐步轉向 Web3，並計劃未來去香港發展。

2. 非常感謝 Jason 老師的分享，那您進入 Web3 的契機是什麼？

畢業前後與在新加坡讀研的同學交流，他那邊的 Web3 氛圍很好；再加上另一位研友，我們一起開始系統做鏈上研究。公司曾經做過區塊鏈板塊（FT 專案，做到國內第二），讓我確信方向有前景，於是決定“跨過去”。

3. 您目前在大廠工作，獲取 Web3 崗位資訊主要靠什麼渠道？

內推 + 線下社群。和同學、同行參加 Web3 線下活動，結識朋友後互相內推。Web3 組織扁平，內推比在招聘平臺“海投”更高效，人也更真誠。

4. 對還沒有人脈、正在讀書的同學，怎麼找到第一份 Web3 技術實習？

兩點：

  • “多實習” 我本科 1 段、碩士 2 段實習經歷。身處蕭條期（疫情後大裁員）仍拿到華為、阿里 offer，實習是最大加分項。
  • “打牢基礎”面試幾乎都考最基礎的計算機知識（組成原理、IO、CPU / 記憶體分片等）。實踐中把這些知識串起來，勝過死背“八股”。

5. 您曾面試國內 Top2 交易所專家崗，面試重點是什麼？

整場一小時幾乎都在問 底層基礎：CPU 分片、記憶體佈局、快取機制、檔案 IO 流程……畢竟 Web2、Web3 都執行在 CPU /伺服器上；理解這些原理，對系統最佳化至關重要。

6. 對想從 Web2 轉 Web3 的開發者，您覺得最大挑戰是什麼？

要分型別：

  • “前端開發者”遷移門檻最低，換包調介面即可。
  • “底層開發者（C/C++ 等）”語言遷移雖快，但更需要 產品思維。Web3 還缺一款真正的“超級應用”，理解業務和生態比單純寫程式碼更重要。

7. AI 時代，程式碼自動生成越來越強，開發者應如何保持競爭力？

大模型可以幫忙改寫或生成程式碼，但 無法替你理解業務場景、做產品決策。領悟產品邏輯、主導系統落地，才是不可替代的核心價值。

8. 對純“小白”如何 0→1 跨入 Web3，您有哪些具體路徑？

  • “商科背景”利用調研與金融知識，學會用 SQL/Superset 等工具做鏈上資料歸因與洞察。
  • “理工背景”已學 C/C++，轉學 Solidity、Rust 相對輕鬆；跟著 GitHub / Ethereum 官方教程做專案。
  • “文科背景”不必硬學程式碼，可發揮 BD、社群運營、內容策劃優勢——Web3 同樣需要市場、媒體、自媒體和展會人才。關鍵是定位自身強項，與行業痛點對接。

9. 您認為大學生想在校期間形成“實習 ↔ 基礎”正向迴圈，應如何落實？

先找一份相關實習 → 真實專案中發現基礎短板 → 回校夯實基礎 → 再申請更有挑戰的實習。每輪都能看見成長與反饋，形成閉環。

10. 如果讓您給想投身 Web3 的在校生或轉型開發者一句話，您會說什麼？

先認清自己的優勢，再把它對映到行業真正缺口上；學習語言只是起點，理解業務才能立足。

11. 訪談接近尾聲，您還有什麼想補充的嗎？

Web3 仍在早期，機會很多。從小處切入、快速試錯，比盲目追風更重要。祝每位同學都能找到適合自己的切口，在鏈上世界開花結果！
