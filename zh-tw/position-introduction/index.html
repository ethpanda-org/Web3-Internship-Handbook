區塊鏈崗位全景圖

約 2920 字大约 10 分鐘

2025-06-12

隨著區塊鏈技術的發展和 Web3 概念的普及，越來越多的企業與開發者開始投身於這一去中心化的未來網路中。但是與傳統網際網路產業不同，Web3 不僅重塑了資料所有權與價值分配的模式，也催生出一系列新的崗位形態，既包括技術端的創新崗位，如智能合約工程師，也涵蓋了產品、運營、社群管理等多種非技術職能。

我們將圍繞 Web3 領域中常見的崗位型別進行詳細介紹，涵蓋前端、後端、智能合約等技術崗位，以及產品運營、社群管理、研究分析等非技術崗位，幫助有志於進入 Web3 行業的同學更清晰地瞭解各類崗位的職責與技能要求。無論你是具備程式設計背景的工程師，還是對產品策略、使用者運營或行業研究更感興趣的同學，都可以在 Web3 找到屬於自己的發展路徑。

想找到合適的區塊鏈實習，我們不要一上來就開始盲目地看網路上那一堆雜亂無章的教程。我們要遵從第一性原理，追根溯源。讓我們從這次學習的終點開始 —— 從 JD（Job Description）開始，瞭解我們要達到一個什麼樣的要求才可以找到一份 Web3 實習。按技術崗和非技術崗來分類，我們先從技術崗的 JD 開始講起。

一、技術崗

1. 前端工程師

  1. 主要職責：

    • 設計和開發基於區塊鏈技術的前端應用，支援去中心化平臺的互動。
    • 透過 React、Vue 等框架實現高效的使用者介面，支援錢包連線、交易簽名、資訊驗證等功能。
    • 整合並最佳化智能合約的前端互動，確保鏈上資料與使用者介面的無縫連線。
    • 與後端團隊協作，基於 GraphQL 或 RESTful API 獲取鏈上和鏈下資料。
    • 持續最佳化前端效能，提升使用者體驗，確保在不同裝置和瀏覽器上的相容性。
    • 關注 Web3 技術趨勢，參與技術評審與分享，不斷最佳化產品架構與程式碼質量。

  2. 職位要求：

    • 本科及以上學歷，電腦科學或相關專業，具備紮實的計算機基礎。
    • 精通 HTML、CSS、JavaScript，熟悉 React、Vue 等前端框架，能夠獨立構建複雜的 UI 介面。
    • 熟悉 Web3.js、Viem、Metamask 等 Web3 技術棧，能夠與智能合約進行互動。（現在 Ethers.js / Web3.js 已經不怎麼使用了，大家現在基本上都是用的 Viem）
    • 瞭解常用的區塊鏈網路，如以太坊、Solana 等，具備 Dapp 開發經驗者優先。
    • 熟悉 Git 版本管理工具，具備良好的程式碼編寫規範，注重程式碼可維護性。
    • 良好的溝通能力和團隊協作精神，能夠在快速發展的環境中高效工作。
    • 具有良好的問題解決能力，能在面對複雜的技術難題時，提出創新的解決方案。
    • 有開源專案或 Web3 相關專案的經驗優先。
    # 常用技術棧
- HTML5
- CSS3
- JavaScript (ES6+)
- React / Vue
- TypeScript
- Next.js
- Ethers.js / Web3.js / Viem

2. 後端工程師

  1. 主要職責：

    • 設計、開發和維護去中心化應用（Dapp）的後端服務，包括鏈上資料互動、智能合約整合和事務處理。
    • 與前端團隊合作，確保前後端資料互動的高效性和穩定性，支援多種 Web3 錢包（如 Metamask）的整合。
    • 基於 Node.js、Python 等技術棧構建高效能的 RESTful 或 GraphQL API，以支援 Web3 平臺的需求。
    • 與智能合約團隊合作，確保智能合約與後端服務的無縫連線，最佳化鏈上資料的讀取和寫入效率。
    • 最佳化後端效能，確保系統的高可用性、高吞吐量和低延遲，滿足高併發訪問需求。
    • 定期進行系統架構和程式碼的評審，不斷提升程式碼質量與技術標準。
    • 參與 Web3 技術的前沿研究，保持對新興區塊鏈技術的學習和應用，推動公司技術迭代。

  2. 職位要求：

    • 本科及以上學歷，電腦科學或相關專業，具備紮實的計算機基礎。
    • 精通 Node.js、Go、Python 等後端開發語言，具有構建高併發、分散式系統的經驗。
    • 熟悉 Viem、Web3.js、Ethers.js 等 Web3 工具，能夠與區塊鏈進行互動並處理鏈上資料。
    • 具備 RESTful API 或 GraphQL 開發經驗，能夠設計高效的 API 服務，支援前端與區塊鏈的互動。
    • 熟悉資料庫技術，具備 MySQL、PostgreSQL 或 NoSQL 資料庫的開發與最佳化經驗。
    • 對智能合約有一定的瞭解，具備與智能合約互動、讀取鏈上資料等相關經驗。
    • 熟悉訊息佇列（如 RabbitMQ、Kafka）及事件驅動架構，能夠處理非同步事務。
    • 熟悉容器化技術（如 Docker、Kubernetes），具備 CI/CD 經驗者優先。
    • 良好的程式碼編寫規範與文件寫作能力，具備良好的團隊協作精神和溝通能力。
    • 有 Web3 專案開發經驗或開源貢獻者優先。
    # 常用技術棧
- Node.js / Go / Python
- Viem / Web3.js / Ethers.js
- RESTful API / GraphQL
- MySQL / PostgreSQL
- Docker / Kubernetes

3. 智能合約工程師

  1. 主要職責：

    • 設計、開發和部署智能合約，確保合約在區塊鏈網路上的安全性、可靠性和高效性。
    • 使用 Solidity 編寫智能合約，涵蓋各類去中心化應用的需求，如 NFT、DeFi、DAO 等。
    • 與團隊合作，定義智能合約的功能需求，並根據需求設計合適的智能合約架構。
    • 部署智能合約至區塊鏈網路（如以太坊、Polygon、Arbitrum、Base 等），並確保合約的高效執行。
    • 編寫智能合約的單元測試，進行程式碼審計與安全性測試，確保合約程式碼無漏洞，避免潛在的安全風險。
    • 最佳化智能合約效能，減少 Gas 費用。
    • 研究和應用最新的區塊鏈技術和智能合約最佳實踐，推動技術的持續進步。
    • 與前端和後端開發團隊緊密協作，確保智能合約與其他系統元件的順暢整合。
    • 為團隊成員提供智能合約相關的技術支援和指導，推動團隊的技術提升。

  2. 職位要求：

    • 本科及以上學歷，電腦科學或相關專業，具備紮實的計算機基礎。
    • 3 年以上智能合約開發經驗，熟練使用 Solidity 或類似的智能合約開發語言。
    • 熟悉 Ethereum、Polygon、Arbitrum、Base 等主流區塊鏈平臺，能夠在這些平臺上部署和維護智能合約。
    • 瞭解智能合約開發的安全性問題，具備智能合約審計和漏洞修復經驗，熟悉常見的攻擊模式（如重入攻擊、溢位、許可權管理等）。
    • 熟悉區塊鏈的基本原理，理解 Gas 費用、交易確認、區塊鏈共識機制等概念。
    • 熟練使用 Foundry、Hardhat、Remix 等智能合約開發框架，具備專案開發、測試與部署經驗。
    • 具備一定的 Viem、Web3.js、Ethers.js 等 Web3 工具使用經驗，能夠與前端或其他系統進行無縫整合。
    • 熟悉 IPFS、NFT、Token 標準（ERC-20、ERC-721、ERC-1155 等）及去中心化身份（DID）等 Web3 相關技術。
    • 具有良好的程式碼編寫規範，注重程式碼的可讀性和可維護性。
    • 良好的溝通能力和團隊協作精神，能夠在快速發展的環境中有效工作。
    # 常用技術棧
- Solidity
- Remix
- Foundry / Hardhat
- Phalcon / Tenderly
- Yul

二、非技術崗

接下來我們來看看非技術崗的工作，適合沒有計算機背景的同學們。

1. 產品與運營

  1. 職位描述：

    • 在 Web3 產品生命週期中負責協調產品釋出、使用者反饋收集和持續改進流程，以提升使用者體驗和產品迭代效率。
    • 執行以使用者獲取、留存和參與度提升為目標的增長戰略，並監控實施效果。
    • 與產品、技術、市場及合規團隊緊密合作，確保產品上市（go-to-market）策略與各部門需求保持一致。
    • 持續分析運營資料，跟蹤關鍵績效指標（KPIs），並根據資料提出最佳化建議。

  2. 職位要求：

    • 熟悉產品上線（Go-to-market）全流程，擅長跨部門資源協調與專案推進。
    • 具備紮實的資料分析能力，能熟練使用 SQL、Excel 或其他資料工具進行資料統計和洞察提煉。

2. 社群管理

  1. 職位描述

    • 構建並管理 Telegram、Twitter（X）、Discord 等社交平臺的社群，實現持續的使用者互動與增長；
    • 組織線上 AMA（問答）、活動、競賽等形式多樣的社群互動，以提升使用者活躍度和品牌粘性；
    • 跟蹤社群健康度指標和情感分析，定期向管理層彙報洞察與最佳化建議；
    • 與營銷團隊協作，制定並執行內容日曆，釋出社群公告和運營手冊。

  2. 職位要求

    • Web3、DAO 或 NFT 社群管理經驗，深刻理解去中心化應用生態；
    • 出色的文案撰寫與溝通能力，能夠有效引導社群討論並快速響應使用者反饋；
    • 熟練使用社群資料分析工具，能夠監測並解讀使用者行為與輿情動態；
    • 具備活動策劃與執行能力，能夠獨立組織線上及線下社群活動。
    # 常用平臺
- Telegram
- Twitter (X)
- Discord

3. 研究分析

  1. 職位描述

    • 收集、整理並分析 Web3 行業市場與使用者資料，編寫可行性研究報告，為產品與運營提供決策支援；
    • 跟蹤區塊鏈協議技術演進及生態動態，撰寫深度研究報告或白皮書；
    • 進行競爭對手分析，評估市場趨勢與使用者行為模式，為戰略規劃提供資料驅動的建議；
    • 支援專案的加密經濟模型設計與博弈論分析，以保證專案的經濟激勵合理性。

  2. 職位要求

    • 熟練使用 Excel、SPSS 或 Python 等資料分析工具，具備定量和定性研究方法經驗；
    • 深入瞭解區塊鏈生態、DeFi 協議及加密經濟學原理；
    • 優秀的報告撰寫與演示能力，能夠清晰傳達研究結論與建議；
    • 精通鏈上資料分析工具（Glassnode、Token Terminal）。
    # 常用工具
- Excel
- SPSS
- Python
- Glassnode
- Token Terminal
