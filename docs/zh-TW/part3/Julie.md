> 在本章，我們走近 ZetaChain 社群負責人 Julie，記錄她 18 歲入圈從翻譯等“Dirty work”起步，借多元內容與社群試驗洞察 VC 與專案增長差異，終掌舵中美社群運營。她強調社群共識即生產力，分享 0→1 冷啟動:任務可感知、簡化敘事、大使激勵、高頻互動；並奉上 AMA 選題結合熱點與覆盤擴散法，以及 00/05 後以長板定位、持續輸出、主動連線、好奇驅動的成長路線。

### 1. 我們邀請到來自 ZetaChain 的社群負責人 Julie Chen 來分享一下她是如何成為 00 後的 Web3 社群增長專家的，Julie 可以先做個簡單的自我介紹嗎？

大家好，我叫 Julie Chen，2018 年、18 歲那年進入幣圈。先後在韓國、新加坡求學，本科主修新聞傳播、國際關係與經濟學，去年畢業於賓夕法尼亞大學碩士專案後加入 ZetaChain，負責華語與北美兩大社群。ZetaChain 主打“一次部署，隨處可用”的多鏈互操作，相容 EVM。業餘我運營推特賬號“@0xjuliechen”，經常主持中英雙語 AMA。

### 2. 您剛剛提到，您從 2018 年就開始了 Web3 職業生涯。據我瞭解您也是從一些可能所謂的 Dirty Work 開始幹起包括內容翻譯，會議籌備等。我想了解的是，從一開始您就從事的是社群運營的相關工作嗎？是什麼樣的契機讓您加入到 Web3 社群運營中來呢？

我在職業早期主動擁抱 **Dirty Work**——從會議記錄、公眾號軟文到 AMA 主持與研報撰寫——這段密集而多元的經歷讓我以極快的速度積累了內容生產與社群運營能力，也讓我意識到“VC 品牌營銷”與“專案方使用者增長”之間的本質差異，並最終促使我跳入真正需要對 **ROI** 負責的一線專案，關鍵體會有三點。

- “**Dirty Work 是加速器**”新聞、Marketing、諮詢、研報輪番上手；我在 2022 年主持 50+ 場中英 AMA，撰寫“我們為什麼投資 XXX”系列，曝光與能力同步成長。
- “**VC 彈性大，專案方更看重 ROI**”VC 可以“燒錢做品牌”，效果對 LP 負責；專案方的每一美元都要拉動真實使用者或開發者，投產比被實時追蹤。
- “**受眾不同導致增長邏輯完全不同**”VC 面向專案方與 LP，重“品牌資產”；專案方面向使用者與社群，重“留存與轉化”，這也是我決定轉向專案一線的核心理由。

### 3. 看來社群運營可能真的是一門“手藝活”，那您認為為什麼社群工作如此重要呢？

市場也在慢慢變風向，原來看專案能不能做起來是看 VC 的背景好不好，但現在是看社群的熱度夠不夠，**做社群是一個專案最關鍵的部分和能力。**

有社群相當於有共識，在去中心化的 Web3，共識就是最寶貴的，說明大家相信你、對你感興趣、並且覺得好玩，這個是很不容易的。

### 4. 您作為一名 00 後，學習的並不是所謂的相關專業，但卻在 Web3，特別是社群運營這一塊依然取得了成功。那我很好奇的是，機遇固然重要，但是我們說個人的努力也很重要，您是怎麼從一位和現在 05 後一樣的在校大學生一步步走到今天，憑藉自己的努力成為 Web3 社群運營專家的呢？

- 本科起就不停試錯，做過十幾份實習——新聞、諮詢、研報、市場……靠做“Dirty Work”找方向。
- 每做一份工作就問自己：喜不喜歡？擅長嗎？能否做得又快又好？
- 長期堅持產出（研報、AMA 主持、雙語內容）讓行業逐漸認識我，機會也隨之出現。

### 5. Julie 方便介紹一下您目前在 ZetaChain 的日常工作是什麼嗎？比如能否和大家分享一下您的 Daily Routine？

- **“華語市場”** 內容教育 + 功能演示；與 Layer 1 / Layer 2、公鏈、大學社團合作 Space / 講座；設計 Galxe / Zealy 任務推動真實鏈上操作；管理 TG / Twitter / Discord 社群。
- **“北美市場”** 主攻線下 IRL ——大會、Meetup、技術分享；與灣區開發者、學術機構聯合駭客松；持續拓展跨鏈部署和 Gas 抽象的開發者激勵。

### 6. 那當時 ZetaChain 社群早期是怎麼從 0 做到第一批核心使用者的？有什麼方法論能和運營小白分享嗎？並且您是如何評估這些方法論的效果的呢？

- **“任務可感知”** 圍繞鏈上真實操作設計任務，而不是單純 RT 轉發。
- **“內容做減法”** 用生活類比例如“一次部署，隨處可用”拆解複雜 Web3 概念。
- **“大使計劃”** 給社群成員明確貢獻通道與激勵。
- **“積極互動”** 表情包 / 周邊大賽、AMA 連麥等增強黏性。

也就是在上述方法下，ZetaChain 社群從 2000+ 人發展到今天的 5000+開發工程師， 人，推特 1 萬人關注；Telegram 群中每個月幾百條訊息到 6K+ 訊息，大家聊市場和幣價、再到聊技術、聊生態，直到把 ZetaChain 中文社群當成嘮家常，分享假期去了哪裡，生態上又有什麼好玩的專案等等。

### 7. 在社群運營過程中，AMA / 線下 Meetup 是常見的運營手段，幾乎每家都在做，似乎已經成為了運營的必需品，作為社群運營專家，那您有什麼寶貴的經驗能夠分享給大家，從而幫助大家獨立的開展一個效果良好的 Meetup / AMA / 活動呢？

我個人認為 AMA 從 21 年興起，現在已經 4 年了，**可能不再是一個最好的傳播途徑和營銷手段，** 不過我還是想分享一些技巧

- **“選題 = 敘事 + 熱點”** 確保主題既能承載專案核心資訊，又是當下社群想聊的。
- **“選對合作方”** 對方願意共同宣傳、共創內容，效果事半功倍。
- **“做覆盤”** 整理 transcript、釋出資料總結貼，形成二次傳播。

### 8. 當資源有限時，您對 00 後 05 後 Web3 從業者的成長路徑有哪些寶貴建議呢？

- **“定位長板”** 先明確自己最擅長什麼，並把它做到極致。
- **“持續輸出”** 運營個人推特、寫觀察與實踐筆記，如“如何理解 X 模型”、“我參與 XX 的經驗”。
- **“主動連線”** 多和專案團隊溝通，給反饋、做貢獻；被看見的前提是先出現。
- **“保持謙遜與好奇”** Web3 最公平，真正的門檻是學習與輸出的持續性，而非資源。
